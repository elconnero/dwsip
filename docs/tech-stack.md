# DWISP Tech Stack

## Frontend (React + Vite)
| Technology | Purpose |
|------------|---------|
| React 18 | UI library |
| Vite | Build tool & dev server |
| TypeScript | Type safety |
| Tailwind CSS | Styling |
| Axios | HTTP client for API calls |
| React Router | Client-side routing |

## Backend (Python + FastAPI)
| Technology | Purpose |
|------------|---------|
| Python 3.11+ | Backend language |
| FastAPI | Web framework |
| SQLAlchemy | Database ORM |
| Pydantic | Data validation |
| Alembic | Database migrations |
| python-jose | JWT authentication |
| passlib | Password hashing |

## Database
| Technology | Purpose |
|------------|---------|
| PostgreSQL | Primary database |
| Neon / Supabase | Hosted PostgreSQL (free tier) |

## External APIs
| Technology | Purpose |
|------------|---------|
| OpenAI API | Process user prompts → game filters |
| RAWG API | Gaming database for recommendations |

## DevOps & Tooling
| Technology | Purpose |
|------------|---------|
| Git | Version control |
| Vercel | Frontend deployment |
| Railway / Render | Backend deployment (free tier) |
| Docker | Containerization (optional) |

## Development Tools
| Technology | Purpose |
|------------|---------|
| pip / poetry | Python package manager |
| npm | Frontend package manager |
| ESLint | Frontend linting |
| Ruff / Black | Python linting & formatting |
